
1、安装指定autojs APK:
adb shell settings put global verifier_verify_adb_installs 0
adb shell settings put global package_verifier_enable 0


adb install -g D:\AndroidProject\ApkTool\autojs.apk
adb shell am force-stop org.autojs.autoxjs


2、把js文件复制到手机对应文件夹下面：(FB_gerenfawen就是需要执行的任务对应的文件名字)
adb push FB_gerenfawen.js /sdcard/Download


3、配置权限：adb shell settings put secure enabled_accessibility_services org.autojs.autoxjs/com.stardust.autojs.core.accessibility.AccessibilityService

adb shell settings put secure accessibility_enabled 1

adb shell appops set --uid org.autojs.autoxjs MANAGE_EXTERNAL_STORAGE allow
adb shell pm grant org.autojs.autoxjs android.permission.READ_EXTERNAL_STORAGE

adb shell pm grant org.autojs.autoxjs android.permission.SYSTEM_ALERT_WINDOW

//截图权限
adb shell appops set org.autojs.autoxjs PROJECT_MEDIA allow


4、开始启动，并传递参数：(FB_gerenfawen就是需要执行的任务对应的文件名字)
adb shell am force-stop org.autojs.autoxjs
adb shell am force-stop com.facebook.katana
adb shell am start -W -n org.autojs.autoxjs/org.autojs.autojs.ui.main.MainActivity --es "net_script_name" "FB_gerenfawen"
